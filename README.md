# finunity-web
O FinUnity Web é a versão web do sistema financeiro FinUnity, desenvolvida utilizando Spring Boot + Thymeleaf, com integração ao MySQL.  Esta versão foi criada com foco em arquitetura MVC, organização de camadas e renderização dinâmica de dados no frontend. Java 17+  Spring Boot  Spring Web  Spring Data JPA  Thymeleaf  PostgreSQL  Maven
